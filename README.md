# vibe-coding-products
<p align="left">
这个仓库展示了一些我通过vbe coding开发的产品项目。

这些项目大多从一个简单的产品想法开始，例如
- 用一个“毒舌黑猫”角色来监督习惯养成，让打卡不再只是机械记录，而是带有更强反馈的体验  
- 在浏览网页时通过一个插件记录当前位置，并像excel一样做查找
- 把现实照片转换为类似《Stardew Valley》的像素风格图像  
- 根据旅行灵感生成可执行的行程规划  

在开发过程中，我使用多种 AI 编程工具来实现功能与界面设计，把想法转化为可使用的应用。

每个项目都尝试从产品角度出发，设计完整的使用流程，并实现核心功能。

</p>

# 主要使用工具

这些项目主要通过 AI 编程工具完成开发。
主要使用工具：

- **Codex CLI**  
- **Claude Code**  
- **Antigravity**  
- **Google AI Studio**  
  用于快速生成前端代码与应用逻辑。
- **Stitch (Google)**  
  AI UI 设计工具，通过自然语言或草图生成完整 UI 界面与前端代码。

## TripMuse 灵感成行
**AI 旅行规划助手**

<a href="https://github.com/zmx0329/TripMuse">

</a>

把“看到的好地方 / 想去的店 / 收藏的攻略”直接变成可执行的旅行行程。 TripMuse 是一个 灵感画布 + 规划面板 的旅行规划助手：左侧像 Pinterest 一样随手贴灵感，右侧像旅行策划师一样帮你做取舍、平衡与落地，最终生成基于真实 POI、动线合理、节奏可控的日程。

你不需要从“去哪儿、怎么玩、先后顺序”开始纠结——只要把你喜欢的内容丢进来，TripMuse 会替你完成从“灵感”到“行程”的收敛过程

## Black Cat Habits  
**毒蛇黑猫监督官 · 习惯养成系统**

<a href="https://github.com/zmx0329/Black-Cat-Habits">

</a>

Black Cat Habits 是一款带有 “毒舌监督人格” 的习惯养成应用。

它不是一个温柔提醒你打卡的工具，而是一只 毒舌、冷静、持续观察你的监督官。
在这里，你的习惯不是被简单记录，而是被持续审视。

当你坚持时，它会认可你。
当你偷懒时，它也会毫不客气地指出来。

这种 人格化监督机制，让习惯管理从“记录行为”变成一种更有情绪反馈的体验。

## Stardew Valley Collection  
**星露谷物集图鉴**

<a href="https://github.com/zmx0329/Stardew-Valley-Collection">
</a>


在《Stardew Valley》的世界里，每一件物品都有属于它的名字与故事。
本项目想把这种温柔带回现实——上传一张照片，系统为它生成像素形态、写下标签与描述，并记录下这一刻的时间。从此，平凡日常也能被珍藏，像素里藏着的是生活本身。
